Customer Churn Prediction

Developed a predictive model to identify customers likely to churn with 89% accuracy, helping reduce churn by 27%.

View Code Live Demo

Project Overview

This project involved developing a machine learning model to predict customer churn for a telecom company. The goal was to identify customers who were likely to cancel their subscriptions, allowing the company to take proactive retention measures.

The dataset contained information on customer demographics, account details, services subscribed, and churn status. After thorough exploratory data analysis and feature engineering, several machine learning models were evaluated, with the Gradient Boosting Classifier achieving the highest accuracy.

The final model achieved 89% accuracy in predicting churn, with a recall of 92% for the churn class (important for identifying as many potential churners as possible). When implemented, this solution reduced actual churn by 27% over six months, saving the company an estimated $1.2M annually.

Key Features

Advanced Feature Engineering

Created meaningful features from raw data like tenure categories, service bundling indicators, and usage patterns that significantly improved model performance.

Class Imbalance Handling

Implemented several techniques including SMOTE, class weights, and stratified sampling to effectively handle the imbalanced dataset (15% churn rate).

Model Interpretability

Used SHAP values to explain model predictions, providing business stakeholders with understandable insights into factors driving churn predictions.

Threshold Optimization

Performed cost-sensitive analysis to determine the optimal prediction probability threshold that balanced accuracy with business objectives.
